## 5. Rules

1. **NAP must match EXACTLY everywhere** — copy-paste from section 1. Same name spelling ("The Makhana"), same website (https://www.themakhana.in with www), same email, primary phone +91 82871 24651 first.
2. **Never pay for links.** Free tiers only. Tupalo's paid verification, Justdial/Sulekha upsells, PR "premium" plans — all skipped.
3. **One submission per site.** No duplicates ever; duplicates hurt NAP consistency. Check the Status column before touching a site.
4. **Track everything here.** Every submission gets a date in Status; once live, paste the live listing/article URL into Status.
5. **Verify dofollow/nofollow at submit time** with browser dev tools — "reported" labels are community consensus, not lab-verified.
6. **Save credentials** (site, login email, password) in the owner's password manager as you go — most sites need the account again for edits.
7. **Content links = value first.** Quora/Reddit answers and guest posts must be genuinely useful; link only where relevant. No drop-links.
